# 🏰 Gothic1 Remake Lockpicker

Веб-приложение для управления и взлома замков с зависимыми ячейками. Система позволяет создавать замки с настраиваемыми связями между ячейками и автоматически находит кратчайший путь к их открытию.

## ✨ Возможности

### Основные функции
- **Управление замками** - создание, редактирование, удаление замков
- **Настраиваемые зависимости** - гибкая настройка связей между ячейками со знаками "+" (туда же) и "-" (противоположно)
- **Автоматический поиск решения** - BFS алгоритм находит кратчайший путь
- **Визуализация решений** - пошаговое отображение с анимацией перемещений
- **Глобальные настройки** - централизованное управление параметрами системы
- **Поиск и пагинация** - удобная навигация по списку замков

### Технические особенности
- 🔍 Регистронезависимый поиск по названиям
- 📱 Полностью адаптивный дизайн
- ⏱️ Защита от бесконечного поиска (таймаут и лимит состояний)
- 💾 Легковесная SQLite база данных
- 🎨 Красивая анимация перемещений

## 🛠 Технологии

- **Backend**: Python 3.8+, Flask
- **Database**: SQLite
- **Frontend**: Bootstrap 5, FontAwesome 6
- **Алгоритмы**: BFS (поиск в ширину)

## 📦 Установка

### Требования
- Python 3.8 или выше
- pip (менеджер пакетов Python)

### Шаги установки

1. **Клонируйте репозиторий**
```bash
git clone https://github.com/yourusername/castle-cracker.git
cd castle-cracker
```

### Запуск и компиляция
1. pip install -r requirements.txt
2. Запустите python app.py
3. Поле успешного запуска перейдите по адресу http://localhost:5000
