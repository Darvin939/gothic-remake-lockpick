Импорт замков

Поддерживаются файлы в формате JSON (один замок или несколько)
Формат файла: 
{
  "version": "1.0",
  "castle": {
    "name": "Название замка",
    "cells": 4,
    "start_positions": [1, 7, 3, 2],
    "dependencies": {
      "1": {"2": "+", "3": "-"},
      "2": {"1": "-", "4": "+"}
    }
  }
}

Можно импортировать несколько замков, используя поле "castles": [...]

Назад
Как экспортировать замки?

Для экспорта замков:

  • Один замок: На странице замка нажмите кнопку Экспорт
  • Все замки: На странице списка замков нажмите кнопку Экспорт все
